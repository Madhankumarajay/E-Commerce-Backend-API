# Spring Boot E-Commerce Backend API

A production-grade E-Commerce Backend API built using Spring Boot, PostgreSQL, JWT Authentication, and Spring Security.

## Features

- JWT Authentication
- Role-based Authorization
- Product Management
- Cart Management
- Order Processing
- Payment Module
- Address Management
- Swagger Documentation

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Spring Security
- JWT
- JPA/Hibernate
- Maven

## Run Locally

1. Clone repository
2. Create PostgreSQL database
3. Configure application.properties
4. Run application

```bash
mvn spring-boot:run
